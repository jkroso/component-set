
# component-set

  add dependencies to component.json

## Getting Started

_With npm_  

	$ npm install component-set -g

## API

```
  Usage: component-set [options] <pkg>

  Options:

    -h, --help         output usage information
    -d, --development  add as a development dependency

  Examples:

    # explicit
    $ component set jkroso/emitter
```
